🔍 Troubleshooting Steps

If you're getting ERR_TUNNEL_CONNECTION_FAILED:
  1. Check Proxy Settings: Disable any proxy or VPN connections
  2. Clear DNS Cache:
    • Windows: ipconfig /flushdns
    • Mac: sudo dscacheutil -flushcache
    • Linux: sudo systemctl restart systemd-resolved
  3. Try Different DNS: Use 8.8.8.8 or 1.1.1.1
  4. Disable Firewall/Antivirus: Temporarily disable to test
  5. Try Different Browser: Test in incognito/private mode
  6. Check Network: Try from different network (mobile hotspot)
